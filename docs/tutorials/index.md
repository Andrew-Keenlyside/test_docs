# Tutorials

These tutorials will guide you through BRIDGE from first principles. Each tutorial is designed to be followed in sequence, building your understanding step by step.

## Getting Started

If you're new to BRIDGE, start here:

| Tutorial | What you'll learn |
|----------|-------------------|
| [Your First Tractography](first-tractography.md) | Run the full pipeline on sample data |
| [Working with Outputs](working-with-outputs.md) | Explore and visualize your results |

## Prerequisites

Before starting these tutorials, ensure you have:

- A working BRIDGE installation (see [Installation Guide](../how-to/install.md))
- Access to a CUDA-capable GPU
- Basic familiarity with Python and command-line tools
